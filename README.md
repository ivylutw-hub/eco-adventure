# 🌍 環保冒險王

**Earth Guardian Adventure｜v5.0 正式發布版**

一款適合國小中高年級的環境教育闖關遊戲。玩家可以自由探索四大冒險關卡，透過答題累積經驗與金幣、修復守護基地，最後解鎖金門特色生物「歐亞水獺」與「鱟」。

## 遊戲特色

- 🗺️ 四大自由選擇的冒險關卡
- 📚 20 個單元、共 200 題固定題目
- 💾 每題即時自動存檔，支援匯出與匯入
- ⭐ Lv.1～Lv.50 守護者成長系統
- 🧑‍🚀 自選玩家名稱、頭像與成就外框
- 🌍 金幣建設「守護基地」
- 🏆 關卡徽章與最高榮耀稱號
- 🦦 Lv.45 解鎖金門歐亞水獺
- 🏝️ Lv.50 解鎖金門鱟與「金門生態傳奇」
- 🔊 可關閉的答題與升級音效
- 📱 支援手機、平板與電腦

## 直接遊玩

下載專案後，開啟 `index.html` 即可遊玩。遊戲不需要伺服器或資料庫。

## 發布到 GitHub Pages

1. 在 GitHub 建立新的 repository。
2. 將這個資料夾內的所有檔案上傳到 repository 根目錄。
3. 開啟 repository 的 **Settings → Pages**。
4. 在 **Build and deployment** 選擇 **Deploy from a branch**。
5. Branch 選擇 `main`，資料夾選擇 `/ (root)`，再按 **Save**。
6. 等候 GitHub 顯示發布網址。

## 存檔說明

遊戲進度預設保存在瀏覽器的 `localStorage`。清除瀏覽器資料可能會移除進度，因此建議定期使用遊戲中的「匯出存檔」備份 JSON 檔案。

## 專案檔案

- `index.html`：遊戲頁面
- `style.css`：介面與響應式設計
- `app.js`：遊戲、存檔、成長及音效邏輯
- `questions.js`：固定題庫
- `manifest.webmanifest`：安裝式網頁設定
- `favicon.svg`：網站圖示
- `social-preview.svg`：社群分享預覽圖

## 隱私

本版本不收集、不傳送玩家姓名或答題紀錄。所有資料預設只保留在玩家目前使用的瀏覽器。
