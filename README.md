# 環保冒險王 v2.0｜自由主題挑戰版

## 本版主要修改
- 修正首頁載入後空白的問題，不再依賴 `UNIT_PLAN`。
- 完全取消 285 個循序單元與解鎖制度。
- 六大主題可自由選擇，每次隨機抽 10 題。
- 今日挑戰會優先抽錯題、未做過的題目，再抽舊題。
- 每題答完自動儲存；未完成挑戰可由首頁繼續。
- 各主題分別保存已答題數、正確率與接觸進度。
- 保留 Google 登入、Firestore 雲端同步、排行榜、角色與錯題本。
- 新增訪客模式，Firebase 暫時無法連線時仍可測試。
- 答錯不標示正確選項；錯題本不直接顯示答案。

## 使用方式
1. 將整個資料夾部署至 GitHub Pages、Firebase Hosting 或其他靜態網站空間。
2. 不建議直接雙擊 `index.html` 測試 Google 登入；請用網頁伺服器開啟。
3. 本機測試可在資料夾執行：`python -m http.server 8000`，再開啟 `http://localhost:8000`。
4. Firebase Authentication 的授權網域需加入實際部署網域。

## 題庫
`questions.js` 保留原始 2,844 題與選項、答案、解析，程式只在執行時依關鍵字自動分類，不修改題目文字。
