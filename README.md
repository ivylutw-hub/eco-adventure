# 環保冒險王 v9.0.2｜GitHub 上傳修正版

這個版本把 CSS、JavaScript 與鱟圖片全部放在網站根目錄，避免 GitHub 網頁上傳時漏掉子資料夾內容，造成畫面只剩白底文字。

## 上傳方式

1. 打開此資料夾。
2. 按 Ctrl+A 全選所有檔案。
3. 將所有檔案直接拖曳到 GitHub 儲存庫根目錄。
4. 確認 GitHub 根目錄直接看得到 `index.html`、`style.css`、`game.js`。
5. Commit changes 後等待約 1–3 分鐘，再用 Ctrl+F5 強制重新整理網站。

不要把 ZIP 直接放進儲存庫，也不要再多包一層資料夾。
