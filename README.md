## v0.2.0 更新
- Google 帳號登入
- 舊匿名使用者可將帳號連結至 Google，保留原 UID 與進度

## Firebase 必要設定
1. Authentication → 登入方式 → 新增供應商 → Google → 啟用。
2. Authentication → 設定 → 授權網域，確認有：
   ivylutw-hub.github.io


## v0.3.0 分級單元規則
- 題庫共 2,844 題，每題只使用一次。
- 初級階段：每單元以初級為主，交替搭配 1 或 2 題中級。
- 初級用完後：改以中級為主，搭配 1 或 2 題中高級。
- 中級用完後：改以中高級為主，搭配 1 或 2 題高級。
- 最後進入高級競賽區，以剩餘高級題為主。
- 每單元原則上 10 題；全題庫最後一單元為剩餘題數。
