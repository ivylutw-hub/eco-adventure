# 環保冒險王 V9.0.1 核心重構測試版

這是由 V8.2 實際原始碼重構的第一個 V9 測試里程碑。

## 本版完成
- 將設定、遊戲核心、Firebase/排行榜拆分為獨立 JavaScript 檔案。
- 保留 2,844 題固定題庫與既有 Firebase 專案。
- 修正「下一題」與「回到關卡」的整塊按鈕可點擊區域。
- 下一題加入防重複觸發保護。
- 加入守護能量資料：答對 +2，答錯不扣。
- 金門鱟改用原創 SVG 圖像，不再只顯示文字。
- 排行榜暫時無法連線時，改為友善離線提示，不影響遊戲與本機積分。
- 套用第一階段溫暖自然的視覺補強。

## GitHub Pages 上傳
將本資料夾內的所有檔案與資料夾完整上傳到儲存庫根目錄，不要只上傳 index.html。

## 測試重點
1. Google 登入與雲端進度。
2. 手機答題後「下一題」是否整顆按鈕都能按。
3. 完成單元後「回到關卡」是否整顆按鈕都能按。
4. Lv.50 金門鱟頭像是否顯示圖片。
5. 排行榜離線時是否仍可正常遊玩。
