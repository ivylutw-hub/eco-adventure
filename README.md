## v0.3.0 分級單元規則
- 題庫共 2,844 題，每題只使用一次。
- 初級階段：每單元以初級為主，交替搭配 1 或 2 題中級。
- 初級用完後：改以中級為主，搭配 1 或 2 題中高級。
- 中級用完後：改以中高級為主，搭配 1 或 2 題高級。
- 最後進入高級競賽區，以剩餘高級題為主。
- 每單元原則上 10 題；全題庫最後一單元為剩餘題數。

## v1.0.0 登入修正
- 完全移除匿名登入與匿名帳號升級流程。
- 僅使用 Google 帳號登入。
- 防止重複點擊造成 `auth/cancelled-popup-request`。
- 登入期間按鈕會鎖定，完成或失敗後才可再次操作。
- 顯示較明確的 Firebase 登入錯誤原因。


## v1.1.0 介面更新

- 首頁改為環保主題遊戲儀表板。

- 保留 Google 登入、分級題庫、金幣、EXP、排行榜與雲端進度。


## v1.1.1 登入修正
- Google 登入由 Popup 改為 Redirect。
- 避免 Edge／Chrome 的 Cross-Origin-Opener-Policy 與重複彈窗問題。
- 所有非登入畫面預設隱藏，避免程式載入異常時同時顯示多個畫面。
- 保留右上角登出按鈕、環保主題首頁與分頁單元。


## v1.2.0 角色功能
- Google 登入後，第一次使用會進入「建立守護者角色」。
- 玩家可以自行輸入守護者暱稱並選擇遊戲頭像。
- Google 大頭貼只用於帳號辨識，不會強制成為遊戲頭像。
- 登入後可點右上角「我的角色」，隨時修改暱稱與頭像。
- 修改後會同步儲存至 Firestore，排行榜與首頁立即更新。
- 保留右上角「登出」按鈕。


## v1.2.1 緊急修正
- 完全移除殘留的 `signInWithPopup()`。
- Google 登入統一使用 `signInWithRedirect()`。
- 修正不存在的按鈕造成 `Cannot read properties of null (reading addEventListener)`。
- 所有可選介面元件改為安全綁定，不再因單一按鈕缺失使整個程式停止。
- 保留 Google 登入後選擇／修改遊戲頭像及登出功能。


## v1.2.2 修正
- 修正首頁已移除 `leaderboardBtn`，但 JavaScript 仍強制綁定該按鈕，造成程式在 Firebase 初始化前中斷。
- 排行榜改由共用 `openLeaderboard()` 函式開啟。
- 所有非必要按鈕改用安全事件綁定，單一介面元素缺少時不會讓整個網站停止。
- Google Redirect 登入、守護者頭像、角色修改與登出功能維持不變。
