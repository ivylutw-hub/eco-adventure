# 環保冒險王 V9.4.1 Beta 5.1.10

這是以 V9.4.0 穩定版為核心製作的第一階段 UI 重構版。

## 部署

解壓縮後，將資料夾內所有檔案上傳到 GitHub Repository 根目錄，讓 `index.html` 位於根目錄。相同檔名直接覆蓋，不需要先刪除舊檔。

## 驗收重點

1. Google 登入正常。
2. 四大題庫關卡可進入並作答。
3. 守護基地可進入、購買、拖曳、鋪路及存檔。
4. 每日簽到正常。
5. 桌機、平板與手機版面可正常排列。


## V9.4.1 Beta 5.1.10
- 玩家成就桌機版調整為每排 5 個。
- 縮小成就卡、圖示、字體與間距，讓五欄排列更清楚。
- 統一所有成就卡的高度、標題區、說明區、進度條與獎勵位置。
- 平板與手機依螢幕寬度自動調整為 4、3、2 或 1 欄。


## Beta 5.1.19
- 放大手機版玩家成就名稱、說明、稀有度與獎勵文字。
- 所在地、天氣、溫度、濕度與 AQI 直接顯示於首頁主視覺，移除獨立外框、底色與陰影。


## V9.4.1 Beta 5.1.21
- 電腦版首頁環境資訊改為緊湊置中排列，縮小項目間距並預留固定高度，避免與其他內容重疊。
- 手機版所在地獨立成一列，天氣、溫度、濕度與 AQI 同列顯示，縮小圖示、文字與整體高度。
- 小尺寸手機加入額外縮放規則，避免首頁主視覺被環境資訊撐大。
