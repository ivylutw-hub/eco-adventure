# CHANGELOG

## V9.2.0

### 管理後臺
- 新增七項儀表板統計：玩家總數、今日登入、近 7 天活躍玩家、全站經驗值、全站金幣、總作答數、今日答題數。
- 新增玩家搜尋與管理，可修改金幣、經驗值、關卡完成進度與管理員權限。
- 新增 Firestore 後臺題庫，可指定 Stage、單元、難度、四個選項、答案與解析，並可啟用、停用或刪除。
- 新增活動管理：雙倍 EXP、每日登入金幣及公告活動，可設定啟用狀態與結束日期。
- 新增排行榜管理：清空目前週榜、由玩家資料重算、匯出可由 Excel 開啟的 CSV。

### 遊戲功能
- 後臺啟用題目會在玩家登入時自動載入，加入指定 Stage／單元；原 2,844 題靜態題庫保留不變。
- 新增活動載入與每日登入獎勵紀錄；雙倍 EXP 會套用於主線及怪獸弱點首次經驗獎勵。
- 玩家文件新增 lastLoginAt、lastActiveAt、totalAnswered、todayAnswered、todayAnsweredDate 統計欄位。
- 怪獸弱點圖鑑新增依 Stage／單元搜尋。
- 「我的守護基地」再次放大，並調整桌面、平板與手機版建設排列。

### 安全與部署
- 更新 firestore.rules：只有 isAdmin=true 的帳號可管理其他玩家、題庫、活動與排行榜。
- 一般玩家仍只能修改自己的玩家文件，且不能自行變更 isAdmin。
