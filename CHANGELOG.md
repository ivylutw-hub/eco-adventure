# CHANGELOG

## V9.3.2

- 修正新版網站尚未部署 `blockedUsers` Firestore Rules 時，登入流程顯示「雲端載入失敗」的相容性問題。
- 停用名單若因舊規則無權讀取，會先略過該檢查並繼續載入原有玩家雲端存檔；部署新版規則後自動恢復完整停用檢查。
- 無活動公告時固定顯示「目前沒有最新活動／祝你今天冒險愉快！」。
- 守護基地恢復 V9.1 的 300px 基本高度，保留拖曳建設與鋪路功能。
- 「編輯基地／自己鋪路／清除路徑」固定放在基地框右上角。

## V9.3.1

- 活動公告無資料時增加「祝你今天冒險愉快！」。
- 新增每日簽到月曆，記錄每月登入日期與全勤進度。
- 當月最後一天完成全月簽到，自動解鎖限定「四季松鼠守護者」。
- 守護基地升級為自由擺設模式，已購建設可拖曳至自訂位置。
- 新增自行鋪路模式，可在草地點擊放置路徑，雙擊單一路徑可移除。
- 基地位置與路徑資料沿用玩家 state，自動同步本機與 Firestore。

# CHANGELOG

## V9.3 療癒管理版

- 每週週期只顯示月／日範圍。
- 玩家管理新增重設遊戲紀錄、指定天數或不限期停用、永久刪除遊戲帳號。
- 新增 blockedUsers 權限資料，停用或刪除的玩家無法進入遊戲。
- 題庫管理新增 CSV／JSON 批次匯入與 CSV 範本下載。
- 首頁改為療癒自然村落風格，加入天空、白雲、小鳥、樹木、花朵與小屋元素。
- 活動管理新增修改、暫停／恢復、首頁公告控制。
- 守護經驗與連續登入的文字位置統一。
- 原每日登入欄位改為活動公告欄。
- 守護基地再次放大。
- 更新 Firestore Rules，僅管理者可操作玩家刪除與 blockedUsers。
