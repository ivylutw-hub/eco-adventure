# 環保冒險王 v7.0｜每週排行榜正式版

## 本版功能
- 使用完整 2,844 題題庫，286 個固定單元。
- 不分班級，所有玩家共用一個每週排行榜。
- 每週一自動開始新的排行週期。
- 首次完成單元：本週積分 +100。
- 該單元首次 10/10：本週積分再 +50。
- 同一單元重玩不會重複增加排行榜積分。
- 排行榜只顯示自訂暱稱、頭像、等級、完成單元與本週積分。
- 遊戲進度仍保存在玩家自己的瀏覽器；Firebase 負責多人排行榜。

## Firebase 啟用步驟
1. 在 Firebase 建立專案。
2. 新增 Web App，將專案設定貼入 `firebase-config.js`。
3. Firebase Authentication 啟用「匿名」登入。
4. 建立 Cloud Firestore 資料庫。
5. 將 `firestore.rules` 內容貼到 Firestore 規則並發布。
6. 將本資料夾內全部檔案上傳 GitHub Pages。

未設定 Firebase 時，遊戲仍可正常單機答題，但排行榜頁會顯示「尚未啟用」。
