# 環保冒險王 v8.0｜雲端穩定正式版

這是目前建議長期使用的穩定版本，已整合：

- 2,844 題完整題庫、286 個固定單元。
- Stage 1～4 依難度固定編排。
- Google 帳號登入與 Firestore 雲端存檔。
- 換手機、平板或電腦後，以同一個 Google 帳號接續進度。
- EXP、金幣、完成單元、怪獸圖鑑、基地、徽章及每週積分同步。
- 每週共同排行榜，不分班級。
- 每單元第一次滿分才取得 1 枚金幣。
- 修正並加強「回到關卡」按鈕事件綁定。
- 保留舊版 v7.0 / v6.0 本機紀錄遷移。
- 離線時先存本機，恢復網路後再同步。

## 上線步驟

1. 將本資料夾內所有檔案覆蓋上傳至 GitHub Pages 儲存庫。
2. Firebase Authentication 必須啟用 Google。
3. Authentication 的授權網域必須包含 `ivylu-hub.github.io`。
4. 將 `firestore.rules` 全文貼到 Firestore「規則」並按「發布」。
5. 上傳後使用 Ctrl + F5 強制重新整理，避免載入舊快取。

## 測試建議

1. 使用 Google 帳號登入。
2. 完成一個單元，返回關卡頁。
3. 關閉頁面後重新開啟，確認進度仍在。
4. 用另一台設備登入同一個 Google 帳號，確認雲端進度載入。
5. 開啟每週排行榜，確認自己的暱稱與積分出現。

## 注意

排行榜分數由前端遊戲計算，適合一般同學間的學習排名；若未來要舉辦正式競賽並防止技術性竄改，需再加入伺服器端驗證。
