# 環保冒險王 v8.2｜排行榜積分同步修正版

本版依照最新規則調整：

- 關卡單元：每題首次答對 +10 EXP。
- 怪獸弱點圖鑑：每題首次答對 +5 EXP。
- 關卡單元：首次達成 10/10，獲得 2 枚金幣。
- 怪獸弱點圖鑑：同一關卡單元收錄的怪獸全部淨化後，首次獲得 1 枚金幣。
- 同一道題、同一單元的獎勵只會發放一次；重玩不重複計分。
- 未曾答對的題目，日後第一次答對時仍可取得該題 EXP。
- 保留 Google 登入、Firestore 雲端存檔、每週排行榜及 2,844 題固定題庫。

部署：解壓縮後將全部檔案覆蓋上傳至 GitHub Pages，完成後按 Ctrl+F5 強制重新整理。
